# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com), and this project adheres to [Semantic Versioning](https://semver.org).

## [0.1.0] - 2026-05-15

### Added

- Initial release of the `wikiaig` agent skill.
- Covers read workflows (search, list, read, cite) with anonymous and read-token auth.
- Covers write workflows (push_page, create_wiki, fork, update_metadata, delete_page) with write-token auth.
- Documents URL patterns, page kinds, content types, HTML page constraints, and HTTP fallback surfaces.
